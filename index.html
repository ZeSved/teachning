Python

What is python? this page of the website will cover the basics of the langauge. These are things that are found in nearly all programming languages.

Variables:

You assign variable by giving them a name, adding an equal sign and then giving the variable a value (as shown below): 


  [insert variable name here] = [insert variable value here]

The types of variables you need to know are:

  • String (text): 
    
      x = 'hello world!'

    When making a string variable, the value HAS to be within single quotes ('') or double quotes (""). If you want to make a string containing something like the word "you're" you just use double quotes to define the value and then you can freely use single quotes inside of it

  • Int/integer (numbers) 
    
      x = 123

    Note that while you can have a value of a variable start with a number or only be a number, you can not have a number at the start of a variable name. Anywhere else in the name is fine, just not as the first letter (a variable also cannot be named numbers).

  • Boolean (True or False) 
    
      x = False
      x = True

    In python, all booleans have to start with a capital letter, idk why but they do.

  • Array (a type of list) 
    
      x = []

    Arrays can be of any size and of any content, the only thing all arrays have in common is that they are defined using square brackets and you seperate the values inside by putting a comma after every value. When you add value into an array you simply add them as if you were to define the variable normally. Arrays can become complicated but these are the types you will use the most:

    Example of a string array:

    
      x = ['this', 'is', 'a', 'string', 'array']

    Example of a number array:

    
      x = [12, 4, 789123, 0323]

    Example of a boolean array:

    
      x = [True, True, False]

    Example of a mixed array:

    
      x = [True, 123, 'hello', 1, False]
  • Missing data (null/undefined) 
    
      x = null
      x = undefined

    You normally only see these variable values if you encounter an error. I am honestly not even sure if you can assign null as a value since 'null' means that the 'item' (variables, functions, etc) doesn't exist and undefined means that the variable has not been assigned a value.

These are the basic (and most used) variables and variable values. These can become advanced, such as putting in object inside of object, or object inside of arrays, or vice versa. They can also have the value of entire 100 line functions. But don't let that scare you because even if you know them really well you would rarely actually use them.

Basic functions:

Functions are reusable code blocks, in this section I will only cover the 3 basic ones:

  • print() 
    
      print()

    print is the most basic function in python, it is also one of the pre-made functions. It is very simple to use, just type print() and within the paranthesis type what you want to print

    Example of printing a string:

    
      print('Hello World!') # Hello world! will show up in the console

    Example of printing a number:

    
      print(5) # 5 will show up in the console

    Example of printing an array:

    
      print(['this', 'is', 'an', 'array', 123, True]) 

      # ['this', 'is', 'an', 'array', 123, True] will show up in the console

    Example of printing a variable:

    
      x = 'hello world!'
      print(x)

      # hello world! will show up in the console

    All variables can be printed and when printed they will show their value (as shown in the example above).

  • len() 
    
      len()

    len() is used to display the length of anything from variables to strings to arrays. You use it by putting whatever you want to know the length of inside its paranthesis (the things inside a function's paranthesis are called parameters).

    Example of printing the number of characters in a string:

    
      print(len('hello'))

      
      # Since the word hello contains 5 letters, 
      5 will be printed to the console.
